MODE
==
Modern Frontend Development Environment

## Requirement
Node.js
https://nodejs.org/ja/

gulp-cli

## Usage
Download
```
$ git clone https://github.com/dragon890811/mode.git dev
```

Move download directory
```
$ cd dev
```

Install
```
$ npm i
```

Start
```
$ gulp
```

If you change script or stylesheet and your browser sync.

Enjoy!
